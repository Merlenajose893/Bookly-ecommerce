## 🛒 Bookly – E-commerce Backend (MERN)

Bookly is a scalable e-commerce backend application built using the MERN stack, designed to handle core online shopping functionalities such as product management, cart operations, and order processing.

### 🔹 Overview
This project focuses on building a robust RESTful API system that supports a complete e-commerce workflow. It demonstrates backend architecture, authentication, and database optimization using MongoDB.

### ⚙️ Tech Stack
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose ODM)  
- **Authentication:** JWT (JSON Web Tokens)  
- **Architecture:** MVC Pattern  

### ✨ Features
- 🛍️ Product management (Create, Read, Update, Delete)
- 🛒 Cart functionality (Add/remove/update items)
- 📦 Order management system
- 🔐 Secure user authentication using JWT
- 🔒 Protected routes with middleware
- ⚡ Optimized database queries for faster response time

### 🚀 Key Highlights
- Built 15+ RESTful API endpoints for e-commerce workflows  
- Implemented JWT-based authentication with token expiry handling  
- Structured backend using MVC architecture for scalability  
- Improved API performance through MongoDB query optimization
